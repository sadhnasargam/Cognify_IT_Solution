# Geospatial Analysis

## Project Overview

This project is a part of the **Cognifyz Technologies Data Science Internship**.

The objective of this task is to analyze the geographical distribution of restaurants using their latitude and longitude coordinates and explore the relationship between restaurant locations and aggregate ratings.

## Dataset

- **Dataset:** Restaurant Dataset
- **File Format:** CSV
- **Source:** Provided by Cognifyz Technologies

## Objectives

The following objectives were completed:

- Explored Latitude and Longitude coordinates.
- Visualized restaurant locations using scatter plots.
- Analyzed the geographical distribution of restaurants.
- Explored the relationship between Latitude and Aggregate Rating.
- Explored the relationship between Longitude and Aggregate Rating.
- Generated visualizations using Seaborn and Matplotlib.

## Technologies Used

- Python
- Pandas
- Matplotlib
- Seaborn
